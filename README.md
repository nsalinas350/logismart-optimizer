# LogiSmart Optimizer 🚚💨

### End-to-End Logistics Optimization Platform
This project is a Fullstack application designed to solve the **Vehicle Routing Problem (VRP)** using mathematical optimization and modern web technologies. It focuses on reducing operational costs and improving resource allocation for logistics fleets.

---

## 🛠️ Tech Stack (The "All-in-One" Approach)
- **Mathematical Engine:** Python (PuLP, Pandas) for Linear Programming.
- **Backend:** Node.js with NestJS & PostgreSQL (Supabase). (Currently with FastAPI)
- **Frontend:** React & Next.js for data visualization. (Currently with Vanilla JS/HTML)
- **Automation:** Integration with Microsoft 365 workflows. (In process)

## 🚀 Key Features
- **Route Optimization:** Heuristic and exact solvers to minimize travel distance.
- **Anomaly Detection:** Statistical auditing of logistics data to identify outliers. (In process)
- **Interactive Dashboard:** Real-time visualization of fleet status and costs.

## 📈 Business Impact
By combining **Operations Research** with **Software Engineering**, this tool aims to:
1. Reduce fuel consumption and carbon footprint.
2. Automate manual reporting tasks, saving hours of operational work.
3. Provide data-driven insights for strategic decision-making.
